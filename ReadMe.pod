perfect-demo - Fully Scripted Terminal Screencasts

=head1 Synopsis

    # Install `perfect-demo`:
    git clone https://github.com/ingydotnet/perfect-demo
    source perfect-demo/.rc

    # Read the manpage:
    man perfect-demo

    # Run a perfect demo:
    perfect-demo $PERFECT_DEMO_ROOT/share/ls-demo.pd

=head1 Description

The C<perfect-demo> command lets you create and run screencastable, terminal-
based demos, with voice annotation... all from a simple text file.

NOTE: Voice requires the C<say> command, usually found on Mac OSX.
